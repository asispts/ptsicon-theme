# ptsicon-theme
> GTK icon theme for private use

## Requirement
Please install one of these fallback icons:
  - [`elementary`](https://github.com/elementary/icons)
  - [`Adwaita`](https://github.com/GNOME/adwaita-icon-theme)

## Added Icons

### Mimetypes
  1. vscode workspace (`*.code-workspace`)
  2. php (`*.php`)
